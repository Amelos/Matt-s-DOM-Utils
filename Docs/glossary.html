Glossary

  • Truthy

    A value that when coerced to a Boolean equals true.

  • Falsey

    A value that when coerced to a Boolean equals false.

  • Host-like Object

    An object resembling a host object, meaning an object that passes one of the following two assertions:

    1. A typeof result of “object” or “function” along with “truthiness”;
    2. A typeof result of “unknown”.

  • Array-like Object

    An object resembling an array, meaning a truthy host-like object with a numeric length property.

  • Node-like Object

    An object resembling a node, meaning a truthy host-like object with a numeric nodeType property.

  • Document Tree

    A parsed tree representing the current document; interfaced with via the DOM; often confused for “the DOM”. 

    document
----doctype
----documentElement
--------head
------------title
----------------text
--------body
------------div
----------------text

  • Subtree

    An object representing a node-like object's recursive childNodes; a tree that's a branch of the document tree. 

    body
----div
--------text
