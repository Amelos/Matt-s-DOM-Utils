Utils.ancestors

Info

A module containing various methods that traverse document tree ancestors.

  1. Filter Methods
  2. Retrieval Methods
  3. Identity Methods

Filter Methods

These methods are “filter” methods, meaning they will return an array-like object of results from each callback invocation. A result of true will collect the object passed; a truthy result will collect the value returned; a falsey result will not be collected. An example follows. 

function handler(
	obj
)
{
	var result = false;
	if (obj && obj.id === "unique") {
		// criterion met
		result = true;
	}
	return result;
}

  • Utils.ancestors.traverse

    Method that traverses a linked list of a node's parent nodes and calls a callback with the current node passed.

    obj
    A node-like object.
    callback
    A function to be used as a callback.

    Return value: Array

Retrieval Methods

  • Utils.ancestors.get

    Method that traverses the parent nodes of a node-like object and shapes them into a linked list.

    obj
    A node-like object.

    Return value: Array || null

Identity Methods

  • Utils.ancestors.is

    Method that checks if a node-like object is an ancestor of another node-like object.

    par
    A node-like object to be checked.
    obj
    A node-like object to be used to generate an ancestral list and searched.

    Return value: Boolean
