Utils.is

Info

A module containing various identity tests utilizing “duck typing”.

  1. Primary Methods
  2. Secondary Methods

Primary Methods

  • Utils.is.hostObject

    Method that determines if an object resembles a host-like object.

    obj
    An object.

    Return value: Boolean

  • Utils.is.arrayLike

    Method that determines if an object resembles an array-like object.

    obj
    An object.

    Return value: Boolean

  • Utils.is.nodeLike

    Method that determines if an object resembles a node-like object.

    obj
    An object.

    Return value: Boolean
