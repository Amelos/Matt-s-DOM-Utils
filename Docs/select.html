  • Utils.select.byName

    Warning: getElementsByName has known issues in various Internet Explorer versions. Utils makes no attempt to correct incorrect behavior for this method; avoid if possible.

    function (doc, name) {

    byName
    Method that wraps getElementsByName, which in turn returns element nodes with a matching name property.
    doc
    A document node-like object.
    name
    A string containing the name to search for.
    return result;

    }

  • Utils.select.byTagName

    function (caller, tag) {

    byTagName
    Method that wraps getElementsByTagName, which in turn returns recursive "children" of the caller with a matching nodeName property.
    caller
    A document node-like object or an element node-like object.
    tag
    A string containing the tag to search for; "*" returns all "children" in the caller's subtree.
    return result;
    Array || null

    }

  • Utils.select.byTagNameNS

    function (caller, local, ns) {

    byTagNameNS
    Method that wraps getElementsByTagNameNS, which in turn returns recursive "children" of the caller with a matching local name and namespace.
    caller
    A document node-like object or an element node-like object.
    local
    A string containing the local name to search for. "*" matches all namespaces in the caller's subtree.
    ns
    A string containing the namespace to search for. "*" matches all local names in the caller's subtree.
    return result;
    Array || null

    }
